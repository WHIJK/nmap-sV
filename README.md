# nmap-sV

Port fingerprint recognition  
**[only tcp  !!!!!]**
```
No input detected. Hint: cat ip:port.txt | file
Usage of nmap-sV:
  -b    Show port banner
  -i    Show all info
  -o string
        Output to  json 
  -thread int
        Threads (default 100)
  -time int
        timeout for port (default 3)

```
